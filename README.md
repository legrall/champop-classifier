Machine learning Classifier for Champop

# Installation

## Get Describable Textures Dataset (DTD)
```(sh)
cd data
wget https://www.robots.ox.ac.uk/~vgg/data/dtd/download/dtd-r1.0.1.tar.gz
tar xf dtd-r1.0.1.tar.gz
rm dtd-r1.0.1.tar.gz
cd ..
```
